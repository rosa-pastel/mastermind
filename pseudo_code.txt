welcome the player, ask if they wanna play a mastermind game
if the answer is yes, game
give brief info about the game rules, colors, clues
play round:
	print empty game space(circles for the code and the clues)
	ask for a guess
	for each guess(circle/number)
		if the guess is in code but in a different place:
			red clue
		if the guess is in code in the right place
			green clue
			increase number of green clues by 1
		else
			no clue
	if number of green clues is 4, tell the player they won
	elsif play round<max number of play rounds, print clues and start the new play round
	else, tell the player they lost, print the code they tried to break, ask if they wanna play again.
